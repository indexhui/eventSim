# 上班前通勤遊戲改進總結

## 主要改進

### 1. 縮短顯示結果時間

- 從 2000ms 改為 1000ms，讓遊戲節奏更快
- 符合上班前快速遊戲的需求

### 2. 放大數值影響

為了讓玩家體驗到因為體力或心情過低而需要請假的狀況，大幅調整了數值影響：

#### 心情影響平衡調整

- **正面影響**：降低到 2-10（原本 8-18），讓正面選項不會過於強勢
- **負面影響**：保持 -8 到 -15，讓負面選項更有影響力
- **整體平衡**：大部分事件的心情扣分比加分多，增加遊戲挑戰性

#### 體力影響放大

- **消耗選項**：從 -1 到 -3 提升到 -2 到 -12
- **恢復選項**：從 1-5 提升到 2-8
- **社交消耗**：熱情聊天可消耗 -12 體力

#### 個性內耗放大

- **討好型人格拒絕**：心情 -12，穩定性 -3，體力 -6
- **優柔寡斷決策**：心情 -8，體力 -4，穩定性 -2
- **容易內耗衝突**：心情 -10，體力 -5，穩定性 -2

### 3. 重新設計通勤事件

專注於上班前的真實場景：

#### 出門階段

- **出門前的猶豫**：時間管理與心情的平衡
- **早餐的選擇**：健康、時間、金錢的取捨

#### 通勤階段

- **地鐵上的搭訕**：社交與拒絕的兩難
- **電扶梯上的爭議**：禮儀與堅持的選擇
- **同事的順風車**：社交壓力與拒絕技巧
- **捷運上的手機聲**：噪音干擾與應對方式
- **月台上的插隊**：正義感與衝突處理
- **到公司前的猶豫**：工作壓力與心態調整

#### 社會性通勤事件

新增 7 個真實的社會性通勤場景：

**噪音相關**

- **後方小孩的哭鬧**：同理心與包容的考驗
- **音樂外放的乘客**：噪音干擾與溝通技巧

**空間與禮儀**

- **開腳的鄰座**：個人空間與禮儀提醒
- **睡著的鄰座**：尷尬情況與體諒選擇
- **佔位的乘客**：公共資源與正義感

**安全與隱私**

- **疑似偷拍**：隱私保護與安全意識

**社會責任**

- **需要座位的老人**：讓座美德與社會責任

### 4. 增強選項與個性的關係

#### 新增個性特質識別

- **討好型人格**：社交傾向低但同理心高
- **容易內耗**：穩定性低且決斷力低
- **優柔寡斷**：決斷力低的玩家

#### 個性特化選項

- **討好型人格**：在拒絕時會出現「勉強接受」選項，但會造成嚴重內耗
- **優柔寡斷**：在需要決策時會出現「再想想」選項，延遲決定反而更累
- **容易內耗**：在壓力情況下會出現「逃避」選項

#### 條件限制系統

某些選項需要滿足特定個性條件才能選擇：

**需要決斷力的選項**

- 直接指出插隊行為：決斷力 ≥ 2，社交傾向 ≥ -2
- 直接拒絕同事順風車：決斷力 ≥ 2
- 直接提醒噪音問題：決斷力 ≥ 2，社交傾向 ≥ -1

**需要社交能力的選項**

- 熱情回應搭訕：社交傾向 ≥ 1
- 欣然接受順風車：社交傾向 ≥ 0

**需要基本決斷力的選項**

- 委婉拒絕順風車：決斷力 ≥ 0
- 換車廂避開噪音：決斷力 ≥ 0
- 直接說不想聊天：決斷力 ≥ 1

#### 內耗機制

- 選項文本會根據個性調整（如「勉強」→「非常勉強」）
- 數值影響會根據個性放大（討好型人格拒絕時心情-5，穩定性-2）
- 視覺提示：內耗選項會顯示橙色「可能內耗」標籤

### 5. 視覺改進

- **個性特化選項**：紫色標籤
- **內耗選項**：橙色標籤
- **快速識別**：玩家可以一眼看出哪些選項可能造成內耗

### 6. 線段式人格特質顯示

重新設計人格特質的視覺化呈現：

#### 設計理念

- **對立標籤**：每個特質都有左右兩個極端標籤
- **中性中心**：中間顯示「中性」作為平衡點
- **視覺指示器**：圓形指示器顯示當前位置
- **顏色編碼**：不同顏色代表不同強度

#### 特質對立標籤

- **專注力**：容易分心 ↔ 高度專注
- **時間感**：時間緊迫 ↔ 精準有餘裕
- **社交傾向**：內向 ↔ 外向
- **決斷力**：優柔寡斷 ↔ 果斷決策
- **好奇心**：保守 ↔ 好奇探索
- **同理心**：自我中心 ↔ 富有同理心
- **穩定性**：情緒波動 ↔ 情緒穩定

#### 時間感特質說明

**定義**：一個人對通勤時間流逝的感受

- **左側（時間緊迫）**：感覺時間不夠用，容易緊張
- **右側（精準有餘裕）**：能準確掌握時間，從容不迫
- **中間（均衡）**：對時間感受平衡
- **初始值**：0（均衡狀態）

#### 視覺特色

- **固定定位**：Header 固定在頂部，滾動時不消失
- **邊框分隔**：底部邊框區分內容區域
- **響應式設計**：適配不同螢幕尺寸
- **簡潔布局**：減少視覺干擾，專注遊戲內容
- **平面設計**：無陰影的簡潔設計風格
- **柔和配色**：灰白背景配合黑灰邊框，視覺更舒適

### 7. UI 界面改進

#### 整體設計調整

- **背景**：使用灰白色（gray.50）背景，比純白色更柔和

- **邊框**：使用更好看的黑灰邊框（gray.300）

#### 視覺特色

- **固定定位**：Header 固定在頂部，滾動時不消失
- **邊框分隔**：底部邊框區分內容區域
- **響應式設計**：適配不同螢幕尺寸
- **簡潔布局**：減少視覺干擾，專注遊戲內容
- **平面設計**：無陰影的簡潔設計風格
- **柔和配色**：灰白背景配合黑灰邊框，視覺更舒適

### 3. 開發者模式

新增開發者模式功能，方便開發和測試：

#### 功能特色

- **切換按鈕**：在事件標題旁顯示 DEV/DEV ON 按鈕
- **影響數值顯示**：開啟後顯示每個選項的具體影響數值
- **條件限制顯示**：顯示選項的條件限制要求
- **視覺化標籤**：用顏色區分正面（綠色）和負面（紅色）影響

#### 開發者體驗

- 快速查看選項影響，無需查看程式碼
- 即時測試條件限制是否正確
- 方便調整數值平衡
- 便於除錯和優化遊戲體驗

## 遊戲設計理念

### 上班前通勤的真實體驗

1. **時間壓力**：出門時間、早餐選擇、到公司時間
2. **社交壓力**：搭訕、同事邀請、拒絕技巧
3. **環境干擾**：噪音、插隊、禮儀衝突
4. **心理狀態**：緊張、猶豫、內耗

### 個性影響機制

1. **討好型人格**：難以拒絕別人，容易內耗
2. **優柔寡斷**：決策困難，延遲決定反而更累
3. **容易內耗**：在衝突和壓力下容易逃避
4. **高決斷力**：能果斷處理問題，但可能顯得強硬
5. **高穩定性**：在壓力下保持冷靜，受到較少負面影響

## 技術實現

### 個性調整系統

- `adjustOptionText()`: 根據個性調整選項文本
- `adjustOptionStats()`: 根據個性調整數值影響
- `addPersonalitySpecificOptions()`: 添加個性特定選項

### 內耗識別

- `isPeoplePleaser()`: 識別討好型人格
- `isProneToBurnout()`: 識別容易內耗的玩家
- 視覺標籤系統：快速識別內耗選項

## 未來改進方向

1. **更多上班前場景**：如選擇交通工具、天氣影響等
2. **個性發展軌跡**：長期遊戲中個性的變化
3. **內耗累積機制**：連續選擇內耗選項的負面影響
4. **社交技能培養**：拒絕技巧、溝通技巧的提升
